## 转换公式

### 垂直尺寸（高度、垂直间距、垂直相关字体）
```swift
固定值 / 852 = 比例值
实际值 = geometry.size.height * 比例值
```

### 水平尺寸（宽度、水平间距）
```swift
固定值 / 393 = 比例值
实际值 = geometry.size.width * 比例值
```

## 优势
✅ 自动适配所有 iPhone 屏幕尺寸
✅ 保持相同的视觉比例
✅ 在小屏幕（iPhone SE）和大屏幕（iPhone Pro Max）上都有良好表现
✅ 代码注释清晰标注了原始固定值
